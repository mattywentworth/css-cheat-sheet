Cheat Sheet: CSS Selectors

Specificity

Selector Example Declaration Description

*

* {

margin: 0;

}

Using an asterisk as your first declared CSS selector will apply the styling in the declaration block to every element in the HTML file. In this case, the margins will be painted as 0, overriding the default browser settings.

html, body

html, body {

color: red;

}

This is an example of a "type" selector. The rule targets all html and body elements and paints them red, overriding the default browser setting.

p

p {

color: olive;

}

Similar to the example above, using a HTML tag by itself as a CSS selector will apply the styling contained in the declaration block to all HTML elements matching the selector. In this case, the CSS will paint all < p > elements in the color Olive, overriding the "color: red;" delcaration established prior.

.class

.class {

font-family: Roboto;

}

By writing a class name with a period at the beginning, you can target all instances of that class with style declarations. In this case, the declaration targets all classes with the name "class" and paints them with the font Roboto, overriding the default browser setting.

#id

#id {

color: goldenrod;

}

By writing an id name with a hash at the beginning, you can target the instance of the id with your style declaration. In this case, the declaration targets the element with the id of "id" and paints them with the font color "goldenrod". So if the element with the id of "id" were a < p > element, this declaration would override the olive color established in the second row of this table.

.class p

.class p {

color: lightskyblue;

}

This is an example of a descendent combinator. By writing the 2 or more selectors with a space between selectors, you can target all elements within other elements. In this case, the rule paints all < p > elements within the element(s) with class "class" as lightskyblue. The < p > elements that are not children of the ".class" class will not be affected.
